# Movie E
A simple Movie Explorer app built with HTML, CSS, and JavaScript. Users can search for movies and view details fetched from the OMDB API.

## Demo
- Live App: [https://<YOUR_USERNAME>.github.io/movie-explorer](https://<YOUR_USERNAME>.github.io/movie-explorer)

## Features
- Search movies by title
- Responsive UI cards showing poster, title, and year
- Link to IMDb for full details

## Installation
1. Clone the repo: `git clone https://github.com/<YOUR_USERNAME>/movie-explorer.git`
2. Navigate into directory: `cd movie-explorer`
3. Obtain an OMDB API key from [OMDB API](http://www.omdbapi.com/apikey.aspx)
4. In `script.js`, replace `YOUR_API_KEY` with your key

## Usage
Open `index.html` in your browser or deploy via GitHub Pages.

## Deployment
1. Push to GitHub: `git push origin main`
2. In repository settings, enable GitHub Pages on `main` branch
3. Access at `https://<YOUR_USERNAME>.github.io/movie-explorer`
